We can:

  1. Divide the internal \([a,b]\) into \(n\) equally-sized subintervals by spacing \(n - 1\) points evenly between \(a\) and \(b\) along the x-axis. (Call these points: \(x\text{,}\) \(x_1\text{,}\) \(x_2\text{,}\) ... \(x_{n-1};\) each interval has a width of \(\Delta x = \frac{b-a}{n}\))

  2. We let \(x_i^*\) be the point in interval \(i\) where we measure the height of the function (consider left, right, or center). The areas of rectangles are: \(f(x_1^*) \Delta x\text{,}\) \(f(x_2^*) \Delta x\text{,}\) ... , \(f(x_n^*) \Delta x\)

  3. The area is approximated as:

    \begin{equation*} A = area(R) \approx f(x_1^*) \Delta x + f(x_2^*) \Delta x + ... + f(x_n^*) \Delta x \end{equation*}
    \begin{equation*} \text{ or } A \approx \sum \limits_{k=1}^{n}f(x_k^*) \Delta x \end{equation*}

  4. We can do better with using more and more rectangles to be more exact:

    \begin{equation*} A = \lim_{n \to \infty} \sum \limits_{k=1}^{n}f(x_k^*) \Delta x \end{equation*}
