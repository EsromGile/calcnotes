Theorem 1.2.4.

Let \(a\text{,}\) \(L_{1}\text{,}\) and \(L_{2}\) be real numbers.

\begin{equation*} \displaystyle\lim_{x \to a} f(x) = L_{1} \mbox{ , } \displaystyle\lim_{x \to a} g(x) = L_{2} \end{equation*}

  1. \(\displaystyle\lim_{x \to a} \left( f(x) + g(x) \right) = \left( \displaystyle\lim_{x \to a} f(x) \right) + \left( \displaystyle\lim_{x \to a} g(x) \right) = L_{1} + L_{2} \)

    Note 1.2.5.
    The limit of of the sum is the sum of the limits.

  2. The above is true also for the difference.

  3. \(\displaystyle \displaystyle\lim_{x \to a} \left( f(x) \cdot g(x) \right) = \left( \displaystyle\lim_{x \to a} f(x) \right) \cdot \left( \displaystyle\lim_{x \to a} g(x) \right)\)

  4. \(\displaystyle\lim_{x \to a} \left( f(x) / g(x) \right) = \left( \displaystyle\lim_{x \to a} f(x) \right) / \left( \displaystyle\lim_{x \to a} g(x) \right)\)

    Note 1.2.6.
    This is true provided that \(\displaystyle\lim_{x \to a} f(x) \neq 0\text{.}\)

  5. \(\displaystyle\lim_{x \to a} \sqrt[n]{f(x)} = \sqrt[n]{\displaystyle\lim_{x \to a} f(x)}\)

    Note 1.2.7.
    This is true provided that \(\displaystyle\lim_{x \to a} f(x) > 0\) if \(n\) is even.
