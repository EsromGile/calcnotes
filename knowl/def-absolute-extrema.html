Definition 3.3.1.

Consider an interval in the domain of a function \(f\) and a point \(x_0\text{.}\)

  • \(f\) has an absolute maximum at \(x_0\) if \(f(x) \leq f(x_0)\) for all \(x\) in the interval.

  • \(f\) has an absolute minimum at \(x_0\) if \(f(x) \geq f(x_0)\) for all \(x\) in the interval.

  • \(f\) has an absolute extrema at \(x_0\) if \(f\) has either of the above at that point.
